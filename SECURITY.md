# Security Policy

## Supported versions

| Version | Supported |
| --- | --- |
| 1.0.x | Yes |

## Reporting a vulnerability

Please report security issues privately before public disclosure.

Preferred order:

1. Use GitHub Security Advisories if they are enabled for the repository.
2. Otherwise contact the repository owner directly through the repository hosting platform.

Please include:

- A short description of the issue
- Reproduction steps or proof of concept
- Expected impact
- Any suggested mitigation

We will triage reports as quickly as possible and coordinate disclosure once a fix is available.
