Launch / Apr 26, 2026

Lexoire launches as an open-source voice-first control surface for coding tools.

Lexoire is built for one core job: keep spoken intent attached to the right repo, provider session, and visible stream of work while coding tools are actually doing things.

What ships in the launch release

  • Voice-first prompting tied to explicit sessions instead of disposable transcripts.
  • Provider routing across Copilot, Claude, and Codex with durable context files.
  • Visible processing state so users can see when the system is listening, working, or done.
  • A cleaner public repo with MIT licensing, docs, CI, and safer release hygiene.

Why the product exists

Coding agents are powerful, but they become harder to trust when voice, session state, and execution visibility live in separate places. Lexoire treats voice as a control surface for real work, not as a novelty transcript.

What comes next

The immediate focus is operational reliability: tighter session flows, cleaner handoffs between providers, and a desktop experience that stays transparent enough to steer during real coding sessions.
