Why it feels stable

The design is intentionally narrow: fewer states, clearer feedback, and a stronger visual hierarchy.

  • Visible state Listening, speaking, and processing all read as separate moments.
  • Shared context Central markdown holds the session story across providers.
  • Fancy graphics Green glows, terminals, and depth cues keep the landing page premium but simple.

What the page should communicate

Speak first. Show the journey. Keep the graphics polished. Make the system feel controlled, not crowded.

Suggested visual direction

Use layered green gradients, a subtle grid, animated terminal cards, and one strong hero focal point.

Open source, not hidden
